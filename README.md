# Portfolio Website

A modern, interactive portfolio website showcasing my frontend development work with smooth animations and 3D elements.

## About

Frontend developer specializing in building responsive, performant web applications with modern frameworks and clean, maintainable code. This portfolio demonstrates my skills in creating engaging user experiences through animation and interactive design.

## Tech Stack

- **React** - UI framework
- **Vite** - Build tool and dev server
- **Three.js & React Three Fiber** - 3D graphics and animations
- **GSAP** - Advanced scroll-based animations
- **TailwindCSS** - Styling and responsive design
- **Iconify** - Icon library

## Features

- 3D animated hero section with interactive planet
- Smooth scroll-triggered animations throughout
- Responsive design for all screen sizes
- Interactive project showcase with hover effects
- Optimized performance and loading

## Installation

```bash
# Clone the repository
git clone https://github.com/Bhaskar-J-Pathak/portfoliowebsite.git

# Navigate to project directory
cd portfoliowebsite

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

## Build for Production

```bash
npm run build
```

The production-ready files will be in the `dist` folder.
